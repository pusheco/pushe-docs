کلید‌های ایمیل#

در این صفحه لیست تمامی کلید‌های اعلان به همراه توضیح آن‌ها قابل مشاهده می‌باشد. قالب کلی ایمیل یک دیکشنری است که کلیدهای زیر را درون خود نگه می‌دارد.

app_ids#

iOS
Android

app_ids

استفاده: اجباری در همه نوع ایمیل
کلید پدر: -
نوع مقدار: رشته string
توضیحات: این کلید بیانگر شناسه اپلیکیشن می‌باشد که برای همه انواع اعلان اجباری می‌باشد
