ШЁШұЪҶШіШЁ (Tag)

ШҙЩ…Ш§ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ъ©Ш§ШұШЁШұШ§ЩҶШӘШ§ЩҶ ШұШ§ ШЁШұЪҶШіШЁ ЪҜШ°Ш§ШұЫҢ Ъ©ЩҶЫҢШҜ. ЩҮШұ ШӘЪҜ ЫҢШ§ ШЁШұЪҶШіШЁ ЩҶЩ…Ш§ЫҢШ§ЩҶЪҜШұ ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ Щ…Ш®ШӘЩ„ЩҒ Щ…Ш®ШӘШө ШўЩҶ Ъ©Ш§ШұШЁШұ Ш§ШіШӘ. ШЁШұШ§ЫҢ Ш§ШұШіШ§Щ„ Ш§Ш№Щ„Ш§ЩҶ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ Ш§ШІ Ш§ЫҢЩҶ ШӘЪҜвҖҢЩҮШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ШұШҜ ЩҲ ШҜШіШӘЩҮвҖҢЫҢ Ш®Ш§ШөЫҢ Ш§ШІ Ъ©Ш§ШұШЁШұШ§ЩҶ ШұШ§ Щ…ШҙШ®Шө Ъ©ШұШҜ.

Щ…Ш«Ш§Щ„:

  • Ъ©Ш§ШұШЁШұШ§ЩҶЫҢ Ъ©ЩҮ ШҜШұ ШӘШ§ШұЫҢШ® Ш®Ш§ШөЫҢ Щ…ШӘЩҲЩ„ШҜ ШҙШҜЩҮвҖҢШ§ЩҶШҜ ШЁШұШ§ЫҢ Ш§ШұШіШ§Щ„ Ш§Ш№Щ„Ш§ЩҶ ШӘШЁШұЫҢЪ© ШӘЩҲЩ„ШҜ.
  • Ъ©Ш§ШұШЁШұШ§ЩҶЫҢ Ъ©ЩҮ ЩҶШ§Щ… ШўЩҶЩҮШ§ Щ…ШӯЩ…ШҜ Ш§ШіШӘ ЩҲ ЫІЫө ШіШ§Щ„ ШҜШ§ШұЩҶШҜ.
  • ЩҲ ...

Ш§Ш¶Ш§ЩҒЩҮвҖҢЪ©ШұШҜЩҶ ШӘЪҜ#

ЩҫШ§ШұШ§Щ…ШӘШұ ЩҲШұЩҲШҜЫҢШ§ШіШӘЩҒШ§ШҜЩҮ
tagsЫҢЪ© ШҙШҰ Ш§ШІ Map Ъ©ЩҮ Ъ©Щ„ЫҢШҜ ЩҲ Щ…ЩӮШҜШ§Шұ ШўЩҶ string Ш§ШіШӘ. Ъ©Щ„ЫҢШҜЩҮШ§ Ш№ЩҶЩҲШ§ЩҶ ШӘЪҜ ЩҲ Щ…ЩӮШҜШ§Шұ ЩҮШұ Ъ©Щ„ЫҢШҜ ЩҮЩ… Щ…ЩӮШҜШ§Шұ ШӘЪҜ ШұШ§ Щ…ШҙШ®Шө Щ…ЫҢвҖҢЪ©ЩҶШҜ

ШҜШұ Щ…Ш«Ш§Щ„ ШІЫҢШұ ШӘЪҜвҖҢЩҮШ§ЫҢ ЩҶШ§Щ…ШҢ ШіЩҶ ЩҲ ШӘШ§ШұЫҢШ® ШӘЩҲЩ„ШҜ ШҜШұ ЩҶШёШұ ЪҜШұЩҒШӘЩҮ ШҙШҜЩҮвҖҢШ§ЩҶШҜ.

  • Java
  • Kotlin
val tags = mapOf(
"name" to "Mohammad",
"age" to "25",
"birthday" to "1435187386"
)
Pushe.addTags(tags);
Map<Strring,String> tags = new HashMap<String, String>();
tags.put("name","Mohammad");
tags.put("age","25");
tags.put("birthday","1435187386");
Pushe.addTags(tags);

ЩҮШұ Ъ©Ш§ШұШЁШұ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШӘШ§ ЫұЫ° ШӘЪҜ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙШҜ

ШӯШ°ЩҒ ШӘЪҜ Ш§ШІ Щ„ЫҢШіШӘ ШӘЪҜвҖҢЩҮШ§#

Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Щ„ЫҢШіШӘ Ъ©Щ„ЫҢШҜвҖҢЩҮШ§ЫҢЫҢ Ъ©ЩҮ Щ…ЫҢвҖҢШ®ЩҲШ§ЩҮЫҢШҜ ШӯШ°ЩҒ Ъ©ЩҶЫҢШҜ ШұШ§ ШЁЩҮ ЩҲШұЩҲШҜЫҢ ШӘШ§ШЁШ№ ШЁШҜЩҮЫҢШҜ.

ЩҫШ§ШұШ§Щ…ШӘШұ ЩҲШұЩҲШҜЫҢШ§ШіШӘЩҒШ§ШҜЩҮ
tagsЩ„ЫҢШіШӘ Ъ©Щ„ЫҢШҜвҖҢЩҮШ§ЫҢЫҢ Ъ©ЩҮ Щ…ЫҢвҖҢШ®ЩҲШ§ЩҮЫҢШҜ Ш§ШІ ШӘЪҜвҖҢЩҮШ§ ШӯШ°ЩҒ Ъ©ЩҶЫҢШҜ
  • Java
  • Kotlin
val tags = listOf("name","age" ,"birthday")
Pushe.removeTags(tags);
List<Strring> tags = new ArrayList<String>();
tags.add("name");
tags.add("age");
tags.add("birthday");
Pushe.removeTags(tags);

ЩҮЩ… ШЁШұШ§ЫҢ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ШұШҜЩҶ ШӘЪҜ ЩҲ ЩҮЩ… ШЁШұШ§ЫҢ ШӯШ°ЩҒ Ъ©ШұШҜЩҶ ШӘЪҜ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ callback ШіШӘ Ъ©ЩҶЫҢШҜ ШӘШ§ ЩҲЩӮШӘЫҢ Ш№Щ…Щ„ЫҢШ§ШӘ Щ…ЩҲЩҒЩӮ ШЁЩҲШҜ Ш§ШІ ШўЩҶ Щ…Ш·Щ„Ш№ ШЁШҙЩҲЫҢШҜ. ШӘЩҲШ§ШЁШ№ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ШұШҜЩҶ ЩҲ ШӯШ°ЩҒ Ъ©ШұШҜЩҶ ШӘЪҜ ЫҢЪ© ЩҶШіШ®ЩҮ ШҜЩҲЩ…ЫҢ ЩҮЩ… ШҜШ§ШұЩҶШҜ Ъ©ЩҮ Ш№Щ„Ш§ЩҲЩҮ ШЁШұ ШӘЪҜвҖҢЩҮШ§ callback ЩҮЩ… ШЁЩҮ Ш№ЩҶЩҲШ§ЩҶ ЩҲШұЩҲШҜЫҢ Щ…ЫҢвҖҢЪҜЫҢШұЩҶШҜ. ШЁЩҮ Ш§Ш¶Ш§ЩҒЩҮ Ъ©ШұШҜЩҶ ШӘЪҜ ШҜШұ Щ…Ш«Ш§Щ„ ШІЫҢШұ ШӘЩҲШ¬ЩҮ Ъ©ЩҶЫҢШҜ.

  • Java
  • Kotlin
val tags = mapOf(
"name" to "Mohammad",
"age" to "25",
"birthday" to "1435187386"
)
Pushe.addTags(tags,object : Pushe.Callback{
override fun onComplete() {
// Tags added
}
});
Map<Strring,String> tags = new HashMap<String, String>();
tags.put("name","Mohammad");
tags.put("age","25");
tags.put("birthday","1435187386");
Pushe.addTags(tags,new Pushe.Callback() {
@Override
public void onComplete() {
// Tags added
}
});

ЪҜШұЩҒШӘЩҶ Щ„ЫҢШіШӘ ШӘЪҜвҖҢЩҮШ§#

Ш®ШұЩҲШ¬ЫҢ Ш§ЫҢЩҶ ШӘШ§ШЁШ№ ЫҢЪ© Map Ъ©ЩҮ ШӘЩ…Ш§Щ… ШӘЪҜвҖҢЩҮШ§ЫҢ Ъ©Ш§ШұШЁШұ ШҜШұ ШўЩҶ ЩҲШ¬ЩҲШҜ ШҜШ§ШұШҜ Ш®ЩҲШ§ЩҮШҜ ШЁЩҲШҜ.

  • Java
  • Kotlin
val tags = mapOf("name" to "Ali")
Pushe.addTags(tags);
val subscribedtags = Pushe.getSubscribedTags(); // tags: {"name":"Ali"}
Map<Strring,String> tags = new HashMap<String, String>();
tags.put("name","Ali");
Pushe.addTags(tags);
Map<String,String> subscribedtags = Pushe.getSubscribedTags(); // tags: {"name":"Ali"}
