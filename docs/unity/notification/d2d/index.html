Ш§Ш№Щ„Ш§ЩҶ Ш§ШІ ЫҢЪ© ШҜЫҢЩҲШ§ЫҢШі ЫҢЩҮ ШҜЫҢЩҲШ§ЫҢШі ШҜЫҢЪҜШұ

Ш§ШұШіШ§Щ„ ЩҫЩҲШҙ Ш§ШІ ЫҢЪ© ШҜШіШӘЪҜШ§ЩҮ ШЁЩҮ ШҜШіШӘЪҜШ§ЩҮ ШҜЫҢЪҜШұ#

Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Щ…ШіШӘЩӮЫҢЩ… Ш§ШІ ЫҢЪ© ШҜШіШӘЪҜШ§ЩҮ ШЁЩҮ ШҜШіШӘЪҜШ§ЩҮ ШҜЫҢЪҜШұ Ш§Ш№Щ„Ш§ЩҶ Ш§ШұШіШ§Щ„ Ъ©ЩҶЫҢШҜ. ШЁШұШ§ЫҢ Ш§ЫҢЩҶ Ш§Щ…Шұ Щ„Ш§ШІЩ… Ш§ШіШӘ Ъ©ЩҮ ЫҢЪ©ЫҢ Ш§ШІ ШҙЩҶШ§ШіЩҮвҖҢЩҮШ§вҖҢЫҢ ШҜШіШӘЪҜШ§ЩҮ Щ…ЩӮШөШҜ ШұШ§ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙЫҢШҜ. ШЁШұШ§ЫҢ Ш§Ш·Щ„Ш§Ш№ Ш§ШІ ШҙЩҶШ§ШіЩҮвҖҢЩҮШ§ЫҢ Щ…ЩҲШ¬ЩҲШҜ ЩҲ ЩҶШӯЩҲЩҮ ШҜШұЫҢШ§ЩҒШӘ ШўЩҶвҖҢЩҮШ§ ШЁЩҮ ШЁШ®Шҙ ШҙЩҶШ§ШіЩҮвҖҢЩҮШ§ ШұШ¬ЩҲШ№ Ъ©ЩҶЫҢШҜ.

ШЁШ§ ШҜШ§ШҙШӘЩҶ ШҙЩҶШ§ШіЩҮвҖҢЫҢ ШҜШіШӘЪҜШ§ЩҮ Щ…ЩҲШұШҜ ЩҶШёШұ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ ЫҢЪ© ШҙЫҢ Ш§ШІ Ъ©Щ„Ш§Ші UserNotification ШЁШіШ§ШІЫҢШҜШҢ ШҜШұ ШўЩҶ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ Ш§Ш№Щ„Ш§ЩҶ ШұШ§ ЩҲШ§ШұШҜ Ъ©ЩҶЫҢШҜ ЩҲ ШЁШ§ ШҜШ§ШҜЩҶ ШўЩҶ ШЁЩҮ ШӘШ§ШЁШ№ PusheUnity.SendNotificationToUser Ш§Ш№Щ„Ш§ЩҶ ШұШ§ Ш§ШұШіШ§Щ„ ЩҶЩ…Ш§ЫҢЫҢШҜ.

ШЁШұШ§ЫҢ ШіШ§Ш®ШӘ ЫҢЪ© ШҙЫҢ Ш§ШІ Ъ©Щ„Ш§Ші UserNotification ШЁШ§ ШҜШ§ШҙШӘЩҶ ЩҮШұЫҢЪ© Ш§ШІ ШҙЩҶШ§ШіЩҮвҖҢЩҮШ§ШҢ Ш§ШІ Щ…ШӘШҜ Щ…ШұШЁЩҲШ·ЩҮ Щ…ЩҲШ¬ЩҲШҜ ШҜШұ Ш§ЫҢЩҶ Ъ©Щ„Ш§Ші ШЁЩҮ ШҙЪ©Щ„ ШІЫҢШұ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ:

UserNotification.WithDeviceId(string androidId);
UserNotification.WithAdvertisementId(string advertisementId);
UserNotification.withCustomId(string customId);

ЩҫШі Ш§ШІ ШіШ§Ш®ШӘ ШҙЫҢ Ш®ЩҲШҜШҢ ШЁШұШ§ЫҢ ЩҲШ§ШұШҜЪ©ШұШҜЩҶ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ Ш§Ш№Щ„Ш§ЩҶ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш§ШІ Щ…ШӘШҜЩҮШ§ЫҢ Ш§ЫҢЩҶ Ъ©Щ„Ш§Ші ШЁЩҮ ШҙШұШӯ ШІЫҢШұ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ:

public UserNotification SetTitle(string title);
public UserNotification SetContent(string content);
public UserNotification SetBigTitle(string bigTitle);
public UserNotification SetBigContent(string bigContent);
public UserNotification SetImageUrl(string imageUrl);
public UserNotification SetIconUrl(string iconUrl);
public UserNotification SetNotifIcon(string notifIcon);
public UserNotification SetCustomContent(string customContent);

Щ…Ш«Ш§Щ„#

ЩҶЩ…ЩҲЩҶЩҮ Ъ©ШҜ ШІЫҢШұ ЩҶШӯЩҲЩҮвҖҢЫҢ Ш§ШұШіШ§Щ„ ЫҢЪ© Ш§Ш№Щ„Ш§ЩҶ ШіШ§ШҜЩҮ ШЁШ§ ШӘЫҢШӘШұ ЩҲ Щ…ШӘЩҶ ШұШ§ ШЁЩҮ Ъ©Ш§ШұШЁШұЫҢ Ъ©ЩҮ ШҙЩҶШ§ШіЩҮвҖҢЫҢ Android ID Ш§ЩҲ ШұШ§ ШҜШ§ШұЫҢЩ… ЩҶШҙШ§ЩҶ Щ…ЫҢвҖҢШҜЩҮШҜ.

using Pushe;
UserNotification notification =
UserNotification.WithDeviceId(androidId)
.SetTitle("ШӘЫҢШӘШұ Ш§Ш№Щ„Ш§ЩҶ")
.SetContent("Щ…ШӘЩҶ Ш§Ш№Щ„Ш§ЩҶ");
PusheUnity.SendNotificationToUser(notification);

ШҜШұ ШөЩҲШұШӘЫҢ Ъ©ЩҮ Щ…Ш§ЫҢЩ„ЫҢШҜ Ш§Ш№Щ„Ш§ЩҶ ЩҫЫҢШҙШұЩҒШӘЩҮвҖҢШӘШұЫҢ ШЁШіШ§ШІЫҢШҜ Ъ©ЩҮ ШЁШ§ ШӘЩҲШ§ШЁШ№ ЩӮШұШ§Шұ ШҜШ§ШҜЩҮ ШҙШҜЩҮ Щ…Щ…Ъ©ЩҶ ЩҶЫҢШіШӘШҢ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Ш§ШІ ШӘШ§ШЁШ№ UserNotification.SetAdvancedNotification Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜ.

ШЁШұШ§ЫҢ Ш§ЫҢЩҶЪ©Ш§Шұ ШЁШ§ЫҢШҜ ШЁЩҮ ШӘШ§ШЁШ№ Ш§Ш·Щ„Ш§Ш№Ш§ШӘ Ш§Ш№Щ„Ш§ЩҶ ШұШ§ ШҜШұ ЩӮШ§Щ„ШЁ ЫҢЪ© JSON String ШЁШҜЩҮЫҢШҜ. ШіШ§Ш®ШӘШ§Шұ JSON ЩҲШұЩҲШҜЫҢ Щ…ШҙШ§ШЁЩҮ ШіШ§Ш®ШӘШ§Шұ Ш§ШіШӘЩҒШ§ШҜЩҮ ШҙШҜЩҮ ШҜШұ Ш§ШұШіШ§Щ„ Ш§Ш№Щ„Ш§ЩҶ ШЁШ§ API Щ…ЫҢвҖҢШЁШ§ШҙШҜ.

ШӘЩҲШ¬ЩҮ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙЫҢШҜ Ъ©ЩҮ Ш§ЪҜШұ Ш§ШІ ШӘШ§ШЁШ№ SetAdvanvedNotification ШЁШұШ§ЫҢ Ш§ЫҢШ¬Ш§ШҜ Ш§Ш№Щ„Ш§ЩҶ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ЩҶЫҢШҜШҢ ШіШ§ЫҢШұ Ш§Ш·Щ„Ш§Ш№Ш§ШӘЫҢ Ъ©ЩҮ ШЁШ§ ШЁЩӮЫҢЩҮвҖҢЫҢ ШӘЩҲШ§ШЁШ№ Ъ©Щ„Ш§Ші ШҜШ§ШҜЩҮ Щ…ЫҢвҖҢШҙЩҲЩҶШҜ ШҜЫҢЪҜШұ ШҜШұ ЩҶШёШұ ЪҜШұЩҒШӘЩҮ ЩҶШ®ЩҲШ§ЩҮЩҶШҜ ШҙШҜ.
