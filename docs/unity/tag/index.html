ШЁШұЪҶШіШЁ (Tag)

Android

ЩҮШұ Ъ©Ш§ШұШЁШұЫҢ Ъ©ЩҮ ШҜШұ Щ„ЫҢШіШӘ ЩҶШөШЁвҖҢЩҮШ§ЫҢ ШЁШұЩҶШ§Щ…ЩҮ ЩӮШұШ§Шұ ЪҜЫҢШұШҜ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ЫҢЪ© Ш¬ЫҢШіЩҲЩҶ ШЁЩҮ ЩҮЩ…ШұШ§ЩҮ Ш®ЩҲШҜ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙШҜ Ъ©ЩҮ ЩҶЩ…Ш§ЫҢШ§ЩҶЪҜШұ ЩҲЫҢЪҳЪҜЫҢвҖҢЩҮШ§ЫҢ Щ…Ш®ШӘЩ„ЩҒ Щ…Ш®ШӘШө ШўЩҶ Ъ©Ш§ШұШЁШұ ШЁШ§ШҙШҜ. ШЁШұШ§ЫҢ Ш§ШұШіШ§Щ„ Ш§Ш№Щ„Ш§ЩҶ ЩҲ ... Щ…ЫҢвҖҢШӘЩҲШ§ЩҶ Ш§ШІ Ш§ЫҢЩҶ ШӘЪҜвҖҢЩҮШ§ Ш§ШіШӘЩҒШ§ШҜЩҮ Ъ©ШұШҜ ЩҲ ШҜШіШӘЩҮвҖҢЫҢ Ш®Ш§ШөЫҢ Ш§ШІ Ъ©Ш§ШұШЁШұШ§ЩҶ ШұШ§ Щ…ШҙШ®Шө Ъ©ШұШҜ.

Щ…Ш«Ш§Щ„:

  • Ъ©Ш§ШұШЁШұШ§ЩҶЫҢ Ъ©ЩҮ ШҜШұ ШӘШ§ШұЫҢШ® Ш®Ш§ШөЫҢ Щ…ШӘЩҲЩ„ШҜ ШҙШҜЩҮвҖҢШ§ЩҶШҜ ШЁШұШ§ЫҢ Ш§ШұШіШ§Щ„ Ш§Ш№Щ„Ш§ЩҶ ШӘШЁШұЫҢЪ© ШӘЩҲЩ„ШҜ.
  • Ъ©Ш§ШұШЁШұШ§ЩҶЫҢ Ъ©ЩҮ ЩҶШ§Щ… ШўЩҶЩҮШ§ Щ…ШӯЩ…ШҜ Ш§ШіШӘ ЩҲ ЫІЫө ШіШ§Щ„ ШҜШ§ШұЩҶШҜ.
  • ЩҲ ...

Ш§Ш¶Ш§ЩҒЩҮвҖҢЪ©ШұШҜЩҶ ШӘЪҜ#

Щ…ШӘШҜ AddTags#

0.4.3
ЩҫШ§ШұШ§Щ…ШӘШұ ЩҲШұЩҲШҜЫҢШ§ШіШӘЩҒШ§ШҜЩҮ
tagsЫҢЪ© ШҙШҰ Ш§ШІ Dictionary Ъ©ЩҮ Ъ©Щ„ЫҢШҜ ЩҲ Щ…ЩӮШҜШ§Шұ ШўЩҶ string Ш§ШіШӘ
using Pushe;
var tags = new Dictionary<string, string> {
{"name","Mohammad"},
{"age", "25"},
{"birthday","1435187386"}
};
PusheUnity.AddTags(tags);

ЩҮШұ Ъ©Ш§ШұШЁШұ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶШҜ ШӘШ§ ЫұЫ° ШӘЪҜ ШҜШ§ШҙШӘЩҮ ШЁШ§ШҙШҜ

ШӯШ°ЩҒ ШӘЪҜ Ш§ШІ Щ„ЫҢШіШӘ ШӘЪҜвҖҢЩҮШ§#

Щ…ШӘШҜ RemoveTag#

0.7.0

ШЁШұШ§ЫҢ ШӯШ°ЩҒ Щ…ЫҢвҖҢШӘЩҲШ§ЩҶЫҢШҜ Щ„ЫҢШіШӘ Ъ©Щ„ЫҢШҜвҖҢЩҮШ§ЫҢЫҢ Ъ©ЩҮ Щ…ЫҢвҖҢШ®ЩҲШ§ЩҮЫҢШҜ Ш§ШІ ЪҶЫҢШіЩҲЩҶ ШӘЪҜ ШӯШ°ЩҒ Ъ©ЩҶЫҢШҜ ШұШ§ ШЁЩҮ ЩҲШұЩҲШҜЫҢ ШӘШ§ШЁШ№ ШЁШҜЩҮЫҢШҜ.

ЩҫШ§ШұШ§Щ…ШӘШұ ЩҲШұЩҲШҜЫҢШ§ШіШӘЩҒШ§ШҜЩҮ
tagsЫҢЪ© ШҙШҰ Ш§ШІ Dictionary Ъ©ЩҮ Ъ©Щ„ЫҢШҜ ЩҲ Щ…ЩӮШҜШ§Шұ ШўЩҶ string Ш§ШіШӘ
// Remove the keys 'name', 'age' and 'birthday' from tags
PusheUnity.RemoveTags("name", "age", "birthday");

ЪҜШұЩҒШӘЩҶ Щ„ЫҢШіШӘ ШӘЪҜвҖҢЩҮШ§#

Щ…ШӘШҜ GetSubscribedTags#

0.4.4

Ш®ШұЩҲШ¬ЫҢ Ш§ЫҢЩҶ ШӘШ§ШЁШ№ ЫҢЪ© Json string Ъ©ЩҮ ШӘЩ…Ш§Щ… ШӘЪҜвҖҢЩҮШ§ЫҢ Ъ©Ш§ШұШЁШұ ШҜШұ ШўЩҶ ЩҲШ¬ЩҲШҜ ШҜШ§ШұШҜ Ш®ЩҲШ§ЩҮШҜ ШЁЩҲШҜ.

using Pushe;
PusheUnity.AddTags(new Dictionary<string, string> {{"name","Ali"}})
Dictionary<string, string> tags = PusheUnity.GetSubscribedTags(); // tags: {"name":"Ali"}
