برای فعالسازی لایبرری در پوشه بایستی ابتدا پلاگین نیتیو را به گریدل اضافه کنید.

اضافه‌کردن MultiDex به AndroidManifest​

AndroidManifest.xml
<application
        android:name="androidx.multidex.MultiDexApplication"
        ...>

راه‌اندازی در Editor​

  • 2019.x
  • 2018.x
  • 2017.x

اضافه‌کردن launcherTemplate​

برای فعالسازی MultiDex بایستی از فایل launcherTemplate.gradle استفاده کنید.

از مسیر Asset/Pushe/Templates/U_2019_4/ فایل launcherTemplate.gradle را به مسیر Assets/Plugins/Android/ انتقال دهید.

فعالسازی پلاگین​

از منوی زیر Resolve را انتخاب کنید تا EDM4U پلاگین را سینک کند.

Assets -> External Dependency Manager -> Android Resolver -> Resolve (Force resolve یا)

نکته

در صورتی که فایل launcherTemplate قبلا موجود بوده کافیست کد multiDexEnabled را به آن اضافه کنید:

android {
    defaultConfig {
        // ...
        multiDexEnabled true
    }

}

اضافه‌کردن mainTemplate​

توجه

در صورتی که هنگام ران‌کردن برنامه گریدل با خطای سینتکس مواجه شد، این خطا به دلیل وجود باگ در Gradle (که در ویندوز رخ می‌دهد) می‌باشد.
برای رفع این خطا پوشه‌ی ExternalDependencyManager را حذف کنید و صرفا فایل mainTemplate را استفاده کنید.
